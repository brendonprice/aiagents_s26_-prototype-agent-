# Mindful Wants & Flourishing Agent

This repository contains a prototype AI agent built for the S26 AI Agents course.  
The agent helps individuals or communities reduce unnecessary wants, counter advertising pressure, and shift toward genuine flourishing.

## What the Agent Does
- Uses warm, non-judgmental Socratic questioning to explore the underlying needs behind a desire to buy something.
- Helps users identify emotional triggers and patterns.
- Reframes advertising and persuasion techniques to reduce their influence.
- Offers flourishing-oriented alternatives (connection, rest, creativity, self-care).
- Encourages gratitude and reflection on what the user already has.

## Tone & Collaboration Style
- Warm coach, not preachy.
- Curious, not directive.
- Acknowledges feelings before asking questions.
- Never shames the user or tells them what to buy.

## Skills (Initial Set)
- `session-start` — routes user intent to the correct skill.
- `want-examination` — explores the desire to buy something.
- `reframe-influence` — identifies persuasion/advertising techniques.
- `flourishing-prompt` — offers 1-, 5-, or 10-minute flourishing alternatives.
- `session-close` — ends the session with grounding or gratitude.

## Purpose of This Repo
This repo is used to:
- Prototype the agent using Claude Code and Entire.
- Iterate on skills and interaction design.
- Store personas, example interactions, and evaluation data.
- Document insights and reflections for the assignment.
